# Pinza accionada por ECG

Pinza robotizada accionada a través de un sensor ECG con integración IoT.

## ¿Qué puedes encontrar en este repositorio?
En el repositorio podrás encontrar:
- Archivos .ino de desarrollo de ESP32
- Diseños en 3D para piezas personalziadas (ajustar escala a la hora de imprimir)
- Presentación con fotografías y explciaciones del proyecto.
